# Philip Bot by Philip Babs

Multi-purpose WhatsApp bot with group admin + personal tools + games.

## Setup
1. Edit settings.js: ownerNumber = 2348165190329
2. npm install
3. npm start
4. Use pairing code from logs to link WhatsApp

## Commands
.kick.promote.demote.warn.mute.unmute.tagall.antidelete
.vv.s/sticker.emojis.del.afk.autoread.autotyping.anticall.report
.guess.tictactoe.ping.owner.help

## Notes
.autoread +.autotyping can trigger WhatsApp warnings if overused.
.anticall cannot block calls - WhatsApp API limitation.
.report shows manual report steps.
